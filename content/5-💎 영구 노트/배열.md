# 배열
- 동일 타입
- 연속 메모리
- 인덱스로 접근

## 기본 연산
- 접근: O(1) 인덱스로 직접 접근
- 탑색 
	- O(n) 선형 탐색, 정렬 안된 배열
	- O(log n) 이진 탐색, 정렬된 배열
- 삽입
	- 끝에 삽입: O(1): 그냥 넣으면 끝
	- 처음에 삽입: O(n): 넣고 하나씩 다 밀어야함
	- 특정 위치 삽입: O(n): 넣고 그 뒤 다 하나씩 밀어야함
- 삭제
	- 끝에서 삭제: O(1) 그냥 삭제하고 끝
	- 처음에서 삭제: O(n) 삭제하고 하나씩 다 밀기
	- 특정 위치에서 삭제: O(n) 삭제하고 그 뒤 하나씩 밀기
